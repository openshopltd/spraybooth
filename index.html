Welcome to the Github site for the Open Shop Spray Booth

We are designing and building a spray booth for the Makerversity workshops and as part of the Open Shop project. A spray booth is important if you are spending any time spray painting with cans or a spray gun. This booth is not designed for polyurethane-based 'two pack' paint systems.

This project has been started by Open Shop Ltd. To contribute you will need a Github account; say hello at oi@openshop.cc and Matt will sort you out.
